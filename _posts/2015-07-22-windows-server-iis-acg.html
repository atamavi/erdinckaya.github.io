 Alınması gereken önlemler; 

1. Windows makinanızda anonymous hesapları kapatın ve yazmayı engelleyin.
2. Windows makinanızda yeni dizin oluşturulmasını engelleyin.
3. Windows makinanızda FTP hizmetini kapatın.
